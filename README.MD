# Boldo Theme

Custom WordPress theme developed with **ACF-powered Gutenberg blocks**, designed for easy content management and flexible layouts.

## Features

- Fully responsive design (desktop, tablet, mobile)
- Custom Gutenberg blocks using Advanced Custom Fields (ACF)
- Modular and clean code structure
- Easy integration for landing pages, blogs, and portfolios
- Optimized for fast performance

## Technologies Used

- WordPress
- PHP
- HTML5 & CSS3
- JavaScript
- Advanced Custom Fields (ACF)
- Gutenberg Blocks
